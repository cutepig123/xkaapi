KAAPI Library

Kernel for Adaptative, Asynchronous Parallel and Interactive programming - A library for ditributed computing

KAAPI Library Manual

All the manual for kaapi is available in the following formats. You can choose one of them to download.

  • HTML (128K bytes) - entirely on one web page.
  • HTML - with one web page per node.
  • HTML compressed (28K gzipped characters) - entirely on one web page.
  • HTML compressed (40K gzipped tar file) - with one web page per node.
  • Info document (24K bytes gzipped tar file).
  • ASCII text (84K bytes).
  • ASCII text compressed (24K bytes gzipped).
  • TeX dvi file (40K bytes gzipped).
  • PostScript file (240K bytes gzipped).
  • PDF file (320K bytes).
  • Texinfo source (4K bytes gzipped tar file).
