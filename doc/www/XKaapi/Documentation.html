XKAAPI means Kernel
for Adaptative, Asynchronous Parallel and Interactive programming.

It is a C++ library that allows to execute multithreaded computation with data flow synchronization between threads. The library is able to schedule fine/medium size grain program on distributed machine. The data flow graph is dynamic (unfold at runtime). Target architectures are clusters of SMP machines.


XKaapi is a MOAIS’ software
